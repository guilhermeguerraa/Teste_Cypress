-- instalar Cypress
npm install cyprtess

-- executar o projeto
npm run cypress open

---------------------
após iniciar cypress clicar em E2E Testing
escolher o navegador e clicar em Start E2E Testing in "Navegador escolhido"
irá aparecer uma lista de arquivos onde deverá clicar para executar o arquivo
---
depois de executado um arquivo, para voltar e escolher outro arquivo para executar 
deverá clicar em Specs no menu a esquera e escolher outro arquivo
---------------------
obs.: na criação da corporação e criação do funcionário não vai salvar e nem sair, pois esse comando está comentado.
Se ocorrer algum erro é só clicar no refresh para recomeçar a execução que irá funcionar normalmente.
